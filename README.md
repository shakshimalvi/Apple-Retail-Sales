## 🍏 Apple Retail Sales Dashboard

This project is an **end-to-end data analytics pipeline** that analyzes Apple retail sales data and presents key business insights through an **interactive Power BI dashboard**.

The workflow includes **data extraction, data cleaning, database integration, SQL querying, and dashboard visualization**.

### 📌 Project Workflow

1. **Data Extraction**

   * Collected Apple retail sales dataset from a website.

2. **Data Cleaning & Transformation**

   * Used **Python and Pandas** to clean missing values, format columns, and prepare structured data for analysis.

3. **Database Integration**

   * Loaded the cleaned dataset into a **SQL database**.
   * Used **SQL queries** to manage and retrieve the required data.

4. **Data Visualization**

   * Connected the database with **Power BI**.
   * Built an **interactive dashboard** to analyze revenue trends, product performance, and category contributions.

### 📊 Dashboard Features

* **Total Revenue by Category**
* **Revenue Contribution by Category**
* **Year-wise Sales Trend**
* **Product-wise Revenue Analysis**
* **Category and Store Filters for interactive analysis**

### 🔍 Key Insights

* **Tablets generated the highest total revenue**, followed by Accessories and Smartphones.
* **Smart Speakers and Streaming Devices have the lowest sales contribution.**
* **2020 recorded the highest overall revenue**, with a decline observed in the following years.
* **Top-performing products include Beats Solo Pro, iPad Pro 11-inch, and MagSafe Charger.**
